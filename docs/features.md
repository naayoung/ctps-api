# CTPS Backend 기능 정리

이 문서는 현재 코드 기준으로 백엔드에서 실제 구현된 기능만 정리합니다.

## 인증

- 이메일/비밀번호 로그인
- 회원가입
- 현재 사용자 조회
- 세션 갱신
- 로그아웃
- 비밀번호 변경
- 비밀번호 재설정 요청 / 확정
- 회원 탈퇴
- 카카오, 구글, GitHub OAuth 시작 / 콜백
